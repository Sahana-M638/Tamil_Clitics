# Tamil Clitic Detection System

## Description

A Python-based Natural Language Processing (NLP) application developed to identify and separate Tamil clitics from words. The system analyzes Tamil Unicode characters using UTF-8 processing techniques and detects clitic suffixes while preserving the root word structure.

## Features

* Tamil word processing
* Clitic detection and separation
* UTF-8 character analysis
* Root word identification
* Tamil text tokenization
* False clitic reduction
* Unicode-based text processing

## Tech Stack

* Python
* Natural Language Processing (NLP)
* UTF-8 Encoding
* Unicode Text Processing
